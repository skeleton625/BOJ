문제 풀이를 완료했습니다.

해당 코드는 다익스트라 알고리즘을 사용해 문제를 풀이했습니다.

해당 문제는 1753 번 문제와 다르게 최종 경로까지 가는 동안 다른 두 점을 먼저 거쳐야 합니다.

문제에선 최단 거리로 접근할 수 없는 케이스가 주어질 수 있기 때문에 두 점을 거쳐 최종 점에 도착하지 못하는 경우도 염두해 두셔야 합니다.

거쳐야할 점을 v1, v2라 가정한다면 두 점을 거쳐 최종 점에 도착할 수 있는 방법은 2가지가 존재합니다.

하나는 시작 점 - v1 - v2 - 최종 점, 둘은 시작 점 - v2 - v1 - 최종 점 입니다.

이 두 가지의 방법을 통해서도 최단 거리를 찾을 수 없을 경우가 완전하게 두 점을 거쳐 최단 거리를 만들 수 없는 경우가 됩니다.

다익스트라 알고리즘 코드는 1753 번의 코드와 동일합니다.
