백준 알고리즘 11401 문제 풀이 중입니다.

문제 풀이를 완료했습니다.

해당 문제는 페르마의 소공식 알고리즘을 사용해 풀이했습니다.

페르마 소공식은 p가 소수이고 a, p가 서로소일 때 a^(p-1)와 p로 나눈 나머지 1이 서로 합동이라는 공식입니다.

여기서 문제의 결과값은 이항계수 C(n,k)를 1,000,000,007로 나눈 나머지입니다.

이항계수 C(n,k) = n!/(k!(n-k)!) 에서 A=n!/k!, B=(n-k)! 라 가정하고 p = 1,000,000,007라 가정하겠습니다.

즉, 저희가 구하게 될 결과값은 C(n,k)%p = AB^-1%p 가 됩니다.

이 떄, B, p에서 p는 소수임을 알 수 있고 B는 정수임을 알 수 있습니다 그러므로 페르마 소공식을 사용해 B^(p-1)%p = 1임을

알 수 있습니다.

즉, 결과적으로 구해야될 식은 C(n,k)%p = (AB^-1*1)%p = (AB^p-2)%p 임을 알 수 있습니다.

해당 문제(이항 계수)의 경우 확장 유클리드, 페르마 소공식 등의 알고리즘을 통해 풀이를 할 수 있습니다.
