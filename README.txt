백준 알고리즘 10159 문제 풀이 중입니다.

문제 풀이를 완료했습니다.

해당 코드는 플로이드 와샬 알고리즘을 사용해 문제를 풀이했습니다.

기본 그래프를 초기화한 다음 플로이드 와샬 알고리즘을 통해 최단거리를 갱신해줍니다. (간선의 비용은 똑같다 생각해 1로 고정)

각 행에서 특정 번호가 연결되어 있을 경우, 그래프의 초기 값이 갱신되므로 이를 통해서 서로 연결되어 있는(비교할 수 있는) 점을

파악할 수 있습니다.

그러므로 연결되어 있는 각 점들의 개수를 세주면 한 점과 비교할 수 있는 모든 점을 파악할 수 있으며 이를 통해서 비교 불가능한 점의

개수를 파악할 수 있게 됩니다. ( 여기서 세어 줄 경우 자기 자신도 세어줄 수 있기 때문에 세어준 값에서 1을 빼줘야 함)
