문제 풀이를 완료했습니다.

해당 코드는 이분 탐색 알고리즘을 사용해 문제를 풀이했습니다.

배열에 들어갈 값을 확인해보면 a[i][j] = i*j인데 이것을 통해서 각 원소 칸에 들어갈 값들은 i의 배수임을 알 수 있습니다.

그러므로 특정 원소 mid를 행 i로 나눈 값보다 작은 숫자가 각 행에 몇개가 있는지 파악하는 것을 통해 mid가 몇 번째

배열 칸에 존재하는지를 파악할 수 있습니다.

이 때 이분 탐색을 위해 left와 right 값을 정의해줘야 하는데, left는 가장 작은 값인 1로 정의하고 right는구해야할 원소

번호 k로 정의해 줍니다. (여기서 k로 right를 정의해주는 이유는 k번째가 될 원소 값은 무조건 k를 넘지 않기 때문입니다.)

이렇게 구한 left와 right로 mid를 구해 해당 mid 뒤에 몇개의 원소값이 존재하는지 파악해 mid의 위치를 파악하고

위치가 k보다 낮을 경우 left를 mid+1로 증가시켜주고 높거나 같을 경우 right를 mid-1로 감소시켜 줍니다.

이 때, 구해야 될 mid 값은 배열에 존재하지 않는 다른 값이 될 수 있기 때문에 right값을 감소시켜 k보다 작아지기 전의

mid 값을 구해야 합니다.

그렇게 되면 k번째에 존재하는 원소의 값을 찾을 수 있습니다.
