백준 알고리즘 2110 문제 풀이 중입니다.

문제 풀이를 완료했습니다..

해당 문제는 이분 탐색 알고리즘을 사용해 풀이했습니다.

문제에서 구해야할 값은 집과 집간의 간격이므로 집 간의 간격을 기준으로 이분 탐색을 구현해 줍니다.

집과 집 간의 거리이므로 가장 작은 값은 1이 되고, 가장 큰 값은 입력값 중 가장 큰 값에서 입력값 중 가장 작은 값을

뺀 값이 됩니다.

이렇게 구한 left, right 값을 기준으로 mid를 구해 mid 거리만큼 떨어진 집에 공유기를 설치합니다.

(여기서 공유기를 설치할 때 mid와 같거나 큰 거리에 설치해야 합니다.)

이 때 설치한 공유기의 개수가 정해진 개수보다 작을 경우, 공유기를 다 설치하지 못했기 때문에

mid 값을 줄여줘야 합니다. (right를 감소시킴)

만일 설치한 경유기의 개수가 정해진 개수보다 크거나 같을 경우 공유기를 설치하고도 거리가 남은 것일 수도 있으므로

mid를 증가시켜 줍니다. (left를 증가시킴) 이 과정에서 최대 값이 결정될 수 있기 때문에 이 때의 mid 값을 max 값과

비교해 따로 저장해 둡니다.

그렇게해서 구한 max값이 해당 문제의 정답이 됩니다.
