문제 풀이를 완료했습니다.

해당 코드는 BFS 알고리즘을 사용해 문제를 풀이했습니다.

성원이와 형석이 둘 다 최선을 다한다고 가정했을 때, 성원이가 이기기 위해선 루트 노드에서 모든 리프 노드까지 가는 깊이의 합이

홀수가 되어야 합니다. (그래야 다음 턴에 형석이가 움직일 말이 없어 지기 때문) 

그렇기 때문에 다음과 같은 방식으로 코드를 구현합니다.

입력값을 가지고 간선들을 연결해 하나의 그래프를 생성한 다음 정점노드 1부터 BFS를 실행합니다.

BFS를 실행하면서 다음 노드로 이동할 때마다 현재 노드의 깊이에 1씩 더해 진행해 줍니다. (정점 노드는 깊이 0)

그렇게 노드를 이동하면서 더이상 이동하지 못하는 때가 될 경우, 리프 노드에 도착한 것이므로 그 때의 깊이들을 각각 더해줍니다.

그렇게 모든 리프 노드의 깊이 값을 더해서 나온 값이 홀수면 성원이가 이기고 짝수면 형석이가 이기게 됩니다.
