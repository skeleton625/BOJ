해당 코드는 이진 검색 트리의 전위순회를 통해 중위 순회를 구하고 이 두 순회를 통해 후위 순회를 출력했습니다.

이진 검색 트리는 부모 노드를 기준으로 왼쪽은 부모 키보다 작은 키값을, 오른쪽은 부모 키보다 큰 키 값을 자식으로 합니다.

이를 통해서 트리의 모든 키 값을 정렬할 경우, 중위 순회의 결과값을 파악할 수 있고 각 노드들의 위치를 파악할 수 있게 됩니다.

(그래서 map에 데이터를 입력하고 데이터를 순서대로 가져옴으로서 정렬했습니다.)

그 다음 전위 순회와 노드들의 위치를 통해 특정 노드를 기준으로 왼쪽과 오른쪽 자식 노드의 범위를 파악할 수 있게 됩니다.

(pos[root] - is 는 현재 부모노드의 왼쪽 노드 개수임 [ pos[root]는 현재 노드 위치, is는 현재 부모 노드의 트리 범위)
