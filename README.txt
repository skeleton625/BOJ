백준 알고리즘 5719번 문제 풀이 중입니다.

문제 풀이를 완료했습니다.

해당 문제는 다윽스타라, BFS 알고리즘을 사용해 문제를 풀이했습니다.

먼저 출발점에 대해 최단 경로를 찾기 위해서 다윽스트라 알고리즘을 실행합니다.

우리가 찾는 것은 최단 경로를 제외한 다음 최단 경로( 거의 최단 경로)를 찾는 것이므로 방금 전에 찾아준 최단 경로에

포함된 도로들을 사용할 수 없더록 수정해줘야 합니다.

이를 위해서 목적지에서 출발지까지 이동하면서 최단 경로에 포함된 도르들을 찾아줘야 합니다.

해당 과정에서는 BFS 알고리즘을 사용해 모든 점을 방문하면서 최단 경로에 포함된 도로인지 파악해 줬습니다.

(이전에 최단 경로를 만든 점들을 연걸해 따로 저장하고 지워줬지만 실패함)

파악한 다음, 최단 경로를 이루는 도로들을 사용하지 못하도록 초기화 해줍니다.

그런 다음 다시 다익스트라 알고리즘을 실행하면 거의 최단 경로를 찾을 수 있습니다.
