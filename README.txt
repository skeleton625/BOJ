백준 알고리즘 2250 문제 풀이 중입니다.

문제 풀이를 완료했습니다.

해당 코드는 dfs 알고리즘을 사용해 문제를 풀이했습니다.

문제에서 두 노드간의 최대 너비를 파악하기 위해선 각 깊이에서 가장 왼쪽에 있는 노드와 오른쪽에 있는 노드를 파악해아 합니다.

그러기 위해선 각 노드들에 대해서 가장 왼쪽에 있는 노드부터 가장 오른쪽에 있는 노드들의 번호를 파악해줘야 합니다.

이를 위해서 dfs 알고리즘을 전위 순회 방식으로 구현했습니다.

전위 순회를 통해 트리의 노드들을 순회하면서 위치를 파악해 주고 각 깊이에서 가장 짧은 위치와 가장 큰 위치를 파악해 줍니다.

이를 파악함으로서 후에 각 깊이에 대한 최대 너비를 알 수 있고 트리에서 같은 깊이에 존재하는 두 노득나의 최대 너비를 파악할 수 있게 됩니다.
