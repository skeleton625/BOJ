백준 알고리즘 13172 문제 풀이 중입니다.

문제 풀이를 완료했습니다.

해당 문제는 페르마의 소공식, 유클리드 호제법을 사용해 풀이했습니다.

면의 개수 n, 모든 면의 합 s, 1,000,000,007 = p라 가정합니다.

이 때, s/n은 해당 주사위를 한 번 굴렸을 때 나올 수 있는 기대값입니다.

여기서 s/n이 기약분수가 될 경우 (s/n)%p의 값을 대신 사용하게 됩니다.

이 때, n은 정수, p은 소수이므로 페르마의 소공식을 적용해 n^(p-1) = 1 ( mod p) 임을 알 수 있습니다.

그러므로 (s/n)$%p = (s*n^(p-2))%p가 됩니다.

이러한 식을 통해서 모든 주사위의 기대값의 합을 구할 수 있습니다.
