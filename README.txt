문제 풀이를 완료했습니다.

해당 문제는 기본적으로 Backtracking을 통해 문제를 해결하는 방법이 존재합니다.

하지만 위의 방법으로 시도할 시 현 문제는 재귀 함수의 실행 횟수가 기하 급수적

으로 늘어나기 때문에 제한시간 내에 해결할 수가 없습니다.

그렇기 때문에 여기서는 다른 방법으로 접근할 필요가 있습니다.

체스판의 퀸으 경유 행, 열, 2개의 대각선으로 움직일 수 있는데, 퀸이 위치한 칸에서

나이트가 움직이는 자리에 다른 퀸이 있을 경우 해당 퀸은 잡을 수가 없습니다.

이렇듯 모든 퀸들이 서로 나이트가 이동하는 위치에 존재하게 될 경우, 다른 퀸들을

잡을 수 없는 N-Queen 문제를 발생시킬 수 있습니다.
