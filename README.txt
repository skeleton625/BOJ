해당 코드는 트리 순회 알고리즘을 사용해 문제를 풀이했습니다.

문제 풀이 관점은 "현재 노드에서 자식 노드까지 가는 가중치는 같아져야 한다." 입니다.

이를 위해서 현재 노드 가중치의 합에 왼쪽, 오른쪽 노드의 가중치 차이만큼 추가로 더해줘야 합니다.

( 이전 노드의 가중치 값 중 작은 값은 그만큼 추가로 높여줘야 왼족, 오른쪽 노드로 가는 가중치 값이 같아지기 때문)

이러한 과정을 dfs 알고리즘으로 진행해, 리프 노드까지 이동한 다음 다시 루트노드로 돌아오면서 가중치 합을 구해줍니다.

그 과정에서 왼쪽, 오른쪽 노드의 가중치 차이와, 현재 노드의 가중치를 계속해서 더해주면 가중치의 최소합을 구할 수

있습니다.
