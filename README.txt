백준 알고리즘 2579 문제 풀이 중입니다.

문제 풀이를 완료했습니다.

해당 코드는 동적 계획법을 사용했습니다.

문제에서 계단을 오를 때 주의해야 할 점은 3가지가 있습니다.

1. 계단은 한 번에 한 계단, 또는 두 계단씩 오를 수 있다.

2. 연속된 세 개의 계단을 모두 밟아서는 안된다. ( 시작점은 계단이 아님 )

3. 마지막 도착 계단은 반드시 밟아야 한다.

3번 연속해서 한칸을 올라갈 수 없으므로 두 칸 연속해서 올라가는 방법과 한 칸과 두 칸을 이동하는 방식을 사용해야 합니다.

또한 최대값을 구하는 방법으로 이전 층의 최대값과 다음 층의 값을 더하는 방식을 통해 최대값을 구할 수 있습니다.

이를 조합하면 다음과 같은 점화식을 얻을 수 있습니다.

층 n 의 최대값 = 층 n-2 의 최대값 + 층 n의 값 (n-2층에서 2칸 올라감)

층 n 의 최대값 = 층 n-3 의 최대값 + 층 n-1의 값(n-3층에서 2칸 올라감) + 층 n의 값(n-1층에서 1칸 올라감)

(n-3의 경우 5층이후부터 연속해서 한 칸씩 올라올 수 있는 경우가 생길 수 있음)

두 개의 점화식 중 최대를 찾아 해당 층의 최대값으로 삼음으로서 마지막 층의 최대값을 찾을 수 있습니다.
