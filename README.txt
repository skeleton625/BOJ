문제 풀이를 완료했습니다.

해당 코드는 BFS 알고리즘을 사용해 문제를 풀이했습니다.

해당 문제는 백준 1167번 문제와 동일한 문제인데 메모리 제한이 이전 문제에서 반이 줄었습니다.

그렇기 때문에 이전 문제처럼 다익스트라 알고리즘을 사용할 경우, 메모리 초과가 발생할 수도 있습니다.

해당 문제는 실질적으로 그래프가 아닌 트리이므로 BFS 알고리즘을 통해 거리를 측정할 경우, 해당 노드까지 가는 거리를 파악할 수 있습니다.
