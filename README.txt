백준 알고리즘 1613 문제 풀이 중입니다.

문제 풀이를 완료했습니다.

해당 코드는 플로이드 와샬 알고리즘을 사용해 문제를 풀이했습니다.

먼저 그래프를 무한대의 값으로 초기화해 줍니다.

해당 문제에서 각 점들의 간선에 대한 비용이 다르다는 얘기는 없으므로 1로 가정하고 무한대의 값(INF)은 모든 점이 연결되었을 경우인

400으로 가정해 줍니다. (이 때, 그래프에서 자기 자신 원소는 0으로 초기화)

그 다음, 그래프의 간선 정보를 입력받고 간선에 대한 비용을 1로 정의해줍니다.

이렇게 입력받은 그래프를 가지고 플로이드 와샬 알고리즘을 실행해 각 점들에 대한 모든 점들의 최단 비용을 계산해 줍니다.

이렇게 최단 비용을 계산한 그래프 배열은 첫 번째 원소 번호(행 번호)는 이전 역사, 두 번째 원소 번호(열 번호)는 이후 역사로 파악할 수

있고 해당 좌표의 원소 값이 초기값이 아닌 갱신된 값일 경우 그 때의 번호들로 전후 관계를 파악할 수가 있게 됩니다.
