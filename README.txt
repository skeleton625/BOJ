백준 알고리즘 7576 문제 풀이 중입니다.

문제 풀이를 완료했습니다.

해당 문제는 너비 우선 탐색과 브루스 포트 알고리즘을 사용해 풀이했습니다.

먼저 현재 들어있는 토마토의 위치를 파악해, queue 자료형 안에 넣어 주면서 안 익은 칸 또한 세어 줍니다.

여기서 queue에 위치를 입력할 때 익은 날짜(깊이) 또한 입력해 줍니다.

queue의 꼭때기에 입력된  토마토들의 위치를 기준으로 너비 우선 탐색을 진행합니다. 

안 익은 칸일 경우 토마토를 익었다는 표시로 1로 변경한 뒤 현재 익은 토마토의 위치 와 꼭대기 토마토의 날짜보다 하루 지난

날짜를 포함패 queue에 입력하고 안 익은 칸의 개수를 1 감소시켜 줍니다.

꼭대기에 입력된 토마토의 날짜가 현재 날짜와 다를 경우 날짜 카운트를 1 증가시켜 줍니다. (시작 날짜는 0)

이런 식으로 queue에 원소가 없을 때 까지 반복하면 진행된 최소 날짜를 파악할 수 있습니다.
