백준 알고리즘 1463 문제 풀이 중입니다.

문제 풀이를 완료했습니다.

해당 코드는 다이나믹 프로그래밍으로 문제를 풀이했습니다.

먼저 작은 값부터 큰 값으로 연산 횟수를 생각해볼 경우, 아래와 같음알 알 수 있습니다.

1. i 숫자는 i - 1의 연산 횟수 + 1

2. i 숫자가 2로 나눠질 경우 i/2의 연산 횟수 + 1

3. i 숫자가 3을 나눠질 경우 i/3의 연산 횟수 + 1

하지만 해당 문제에선 연산의 최소 횟수를 파악하는 문제이므로 2, 3번의 경우 기존 i 숫자의 연산 횟수와 비교해 최소값으로 입력해줘야 합니다.

그리고 2, 3번의 경우 2로도 나눠질 수 있고 3으로도 나눠질 수 있기 때문에 다른 조건으로 최소값을 파악해줘야 합니다.

이 과정을 입력 숫자까지 반복하면 입력 숫자의 최소 연산 횟수를 파악할 수 있습니다.
