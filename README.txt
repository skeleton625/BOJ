해당 문제에서 색의 비용을 최소로 하기 위해서는 사용하는 색의 개수를 최소한으로 해야합니다.

이것을 보았을 때, 색의 개수에 따라 배용을 최소로 할 수 있는 노드 개수는 정해져 있다는 것을 알 수가 있습니다.

ex) 색 1개 -> 노드 개수 1개, 색 2개 -> 노드 개수 2개

해당 점화식S(n)은 파악해 봤을 때 다음과 같음을 알 수 있습니다.

s(n) = 2^(n-1) / n = 색의 수, S(n) = 노드 개수

문제에서 사용할 수 있는 최대 노드 개수는 100,000 개이므로 최소 비용을 가지기 위한 색의 수는 

2^16 < 100,000 < 2^17로 n = 18 임을 알 수 있습니다.

그러므로 최소로 사용해야할 색의 개수가 18개임을 파악하고 각 색을 택했을 때, 최소 비용이 어떻게 되는지를 구하면 됩니다.
