백준 알고리즘 1753 문제 풀이 중입니다.

문제 풀이를 완료했습니다.

해당 코드는 다익스트라 알고리즘을 사용해 문제를 풀이했습니다.

각 점들의 연결 정보와 간선의 비중 값을 입력 받습니다.

이 때, 점 하나에 여러 개의 점이 연결될 수 있으므로 점당 한 개의 배열이 필요합니다.

(배열의 원소 하나에 연결된 점의 번호와 해당 간선의 비중 값이 들어감)

그 다음 우선순위 큐(다익스트라 알고리즘을 실행하기 위한 자료 구조)에 시작 점의 위치와 비중(시작점이므로 0)을 넣어줍니다.

이후 다음과 같은 과정을 반복해 줍니다.

1. 우선순위 큐에 꼭대기 간선 정보를 가져옵니다.

2. 해당 점의 최소 거리보다 간선 정보에 저장된 최소거리가 길 경우 우선순위 큐의 다음 꼭대기 간선 정보를 가져옵니다.

3. 2 번이 아닐 경우 현재 간선 정보에 포함된 점과 연결된 점들을 순회합니다.

4. 연결된 각 점들을 순회하면서 현재 점의 최소 거리과 간선 비중 합이 순회한 점의 현재 최소 거리보다 작을 경우 갱신하고

우선 순위 큐에 입력해 줍니다.

5. 위 과정을 우선순위 큐에 값이 없어질 때까지 반복해 줍니다.

위의 알고리즘을 실해할 경우, 시작점으로부터 각 점까지 이동하는 최소 거리를 파악할 수 있습니다.
