백준 알고리즘 1655 문제 풀이 중입니다.

문제 풀이를 완료했습니다.

해당 코드는 최대힙, 최소힙을 사용해 문제를  풀이했습니다.

문제에서 구하기 원하는 값은 입력된 데이터 중, 중간 값이므로 중간을 기준으로 작은 값과 큰 값으로 나눠야 합니다.

그렇게 하기 위해선 최대힙에는 중간보다 작거나 같은 값, 최소힙에는 중간보다 큰 값이 들어가도록 구현하면 됩니다.

그렇게 될 경우 최대힙의 꼭대기 값이 현재 입력된 값들의 중간 값이 되게 됩니다.

이 때, 이 알고리즘이 유지되기 위해선 항상 최대 힙과 최소 힙의 크기가 1 차이 나거나 같아야 하므로 최대힙과 최소힙에

번갈아 가며 입력값을 입력합니다.

입력값을 입력할 때, 최대 힙의 꼭대기 값이 최소 힙의 꼭대기 값보다 클 경우 두 꼭대기 값을 바꿔줘야 합니다.
