백준 알고리즘 2875 문제 풀이 중입니다.

문제 풀이를 완료했습니다.

해당 코드는 그리디 알고리즘을 사용해 문제를 풀이했습니다.

문제는 인턴십 지원자를 제외하고 남자 2명, 여자 1명으로 이루어진 팀이 최대가 되는 방법을 구하는 것입니다.

그러기 위해서는 여자의 수를 최소한으로 줄이고 팀을 구성할 수 없는 남성들을 최대한 인턴십으로 빼는 것입니다.

즉, 남성 수를 2로 나눈 나머지가 없으면서 나눈 몫이 여성 수 보다 적을 경우, 여성을 인턴십에 보내고

그렇지 않은 경우엔 남성을 인턴십에 보냅니다.

그렇게 보낸 뒤, 남은 남성과 여성으로 팀을 구성하게 되면 최대 팀 개수를 구할 수 있게 됩니다.
