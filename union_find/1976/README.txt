백준 알고리즘 1976 문제 풀이 중입니다.

해당 코드는 Union Find 알고리즘을 사용해 문제를 풀이했습니다.

N * N 행렬로 주어지는 도시 연결 정보를 통해 트리를 생성해 줍니다.

여기서 Union Find  알고리즘을 통해 두 트리(집합)을 연결해 줄 때, 순환 그래프가 되지 않도록 이미 연결된 간선의 정보들

( 현재 행 번호보다 작은 열 번호에 대해 간선 정보를 가진 것)은 무시해 줍니다.

그렇게 완성된 트리들에 포함된 노드(여행지)들은 모두 서로 순회가 가능하므로 루트 노드가 같으면 이동이 가능해 집니다.

그러므로 Union Find 알고리즘을 통해 트리를 완성하고 주어진 경로들의 루트노드를 확인해 만일, 루트노드가 하나라도 다른

것이 있을 경우, 경로로 채택할 수 없는 것이고, 루트노드가 모두 같을 경우, 여행 경로로  채택할 수 있게 됩니다.
