문제 풀이를 완료했습니다.

해당 코드는 BFS, 위상 정렬 알고리즘을 사용해 문제를 풀이했습니다.

문제에서 한 작업을 수행하기 위해선 그 작업과 연결되어 있는 다른 작업을 먼저 수행해야 합니다.

이 때, 이전에 처리해야할 작업들 중 더 오래 걸리는 작업의 시간을 현재 작업의 이전 처리 시간으로 정해줘야 합니다.

이는 시간이 오래 걸리는 작업을 처리하면서 동시에 시간이 적게 걸리는 작업을 수행하기 위해서 입니다.

문제의 예시를 예로 들 경우, 6번을 수행하기 위해서 2번과 4번 작업을 우선적으로 처리해야 합니다.

여기서 2번 작업을 수행하는데 걸리는 시간은 6이고 4번 작업을 수행하는데 걸리는 시간은 11입니다.

2번 작업을 수행하고 난 뒤에도 4번 작업이 끝날 때 까지 기다려야 하기 때문에 6번을 시작하기 위해선 적어도 시간이 11 필요합니다.

그러므로 이를 통해 문제를 처리할 경우, 마지막 문제를 처리할 때까지 걸리는 최소 시간을 파악할 수 있습니다.
