문제 풀이를 완료했습니다.

해당 코드는 완전 탐색을 이용해 문자열에 존재하는 "nemo"라는 문자열을 찾았습니다.

이번엔 string.h 라이브러리에 존재하는 strstr 함수를 사용해 문자열 내부에 존재하는 다른 문자열을 찾는 방법을 통해 문제를 해결했습니다.

다만 "nemo"라는 문자열만을 판별할 수 있도록 입력된 문자열의 대문자를 전부 소문자로 바꿔주는 작업을 추가했습니다.

( strstr -> 문자열 내에서 특정 문자열을 찾는 함수, strchr -> 문자열 내에서 특정 문자를 찾는 함수 )
