문제 풀이를 완료했습니다.

해당 코드는 다이나믹 프로그래밍과 캐시를 사용해 문제를 풀이했습니다.

문제에서 다리를 건설할 때, 한 다리는 다른 다리와 교차해서 건설할 수 없으며 왼쪽에 있는 n개의 모든 점에 대해서

무조건 오른쪽과 연결되어야 합니다.

n = 1 이라고 생각했을 때, 오른쪽 점이 개수(m)가 1 ~ 30일 경우, 다리의 개수는 당연하게 각각 1 ~ 30이 됩니다.

그렇다면 n = 2이고 m = 3일 때를 생각해 봅시다.

여기서 왼쪽 첫 번째 점과 오른쪽 첫 번째 점을 연결한다고 가정하면 남는 점은 n = 1, m = 2가 되므로

다리를 건설하는 경우는 2가지가 됩니다.

그리고 왼쪽 첫 번째 점과 오른쪽 두 번째 점을 연결한다고 가정하면 연결 가능한 점은 n = 1, m = 1가 되므로

다리를 건설하는 경우는 1가지가 됩니다.

즉, 이것을 점화식으로 나타낼 경우 [2][3] = [1][2] + [1][1]가 됩니다.

마찬가지로 n = 3, m = 4일 때의 점화식을 생각해보면 [3][4] = [2][3] + [2][2]가 된다는 것을 알 수 있습니다.

그러므로 점화식은 [a][b] = [a-1][b-1] + ... + [a-1][a-1] 가 됩니다.
