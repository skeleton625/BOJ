백준 알고리즘 11758 문제 풀이 입니다.

문제 풀이를 완료했습니다.

해당 문제는 CCW, 즉 두 선분의 외적을 사용해 문제를 풀이했습니다.

문제에서 입력으로 세 점이 주어지고 문제의 해답은 그 점들을 순서대로 연결한 선분의 방향을 구하는 것입니다.

여기서 세 점들을 순서대로 연결하게 될 경우, 두 선분임을 알 수 있는데

여기서 입력된 점이 순서대로 A, B, C라 가정할 경우 두 선분 백터는 AB, BC로 나타낼 수 있습니다.

그러므로 두 선분 백터 AB, BC는 백터의 외적을 통해 두 선분이 어느 방향으로 가르키고 있는지를 파악할 수 있게 됩니다.

참고로 두 백터 A(a1,a2,a3) B(b1,b2,b3)일 경우 A, B의 외적은 다음과 같다

A x B = (a2b3 - a3b2, a3b1 - a1b3, a1b2 - a2b1)
