백준 알고리즘 2166 문제 풀이 중입니다.

문제 풀이를 완료했습니다.

해당 문제는 CCW 알고리즘, 즉 백터간의 외적을 사용해 문제를 풀이했습니다.

해당 문제는 n개의 점이 주어지고 n개의 점으로 이루어진 다각형의 넓이를 구하는 것입니다.

여기서 다각형의 넓이는 점(백터)간의 외적을 통해 구할 수 있습니다.

점들이 (x[1], y[1]) , (x[2], y[2]) ... (x[n], y[n]) 같다고 가정할 때 다각형의 넓이 식은 다음과 같습니다.

1/2 * |(x[i]+x[i+1])*(y[i]-y[i+1])|

모든 i를 통해 구한 값을 합쳐줄 경우, 다각형의 넓이를 구할 수 있게 됩니다.
