백준 알고리즘 14002 문제 풀이 중입니다.

문제 풀이를 완료했습니다.

해당 코드는 이분 탐색을 사용한 LIS 알고리즘을 사용해 문제 풀이를 완료했습니다.

기존에 사용한 LIS 알고리즘을 사용하는 경우, 정확한 증가하는 부분 수열을 파악할 수가 없습니다.

예를 들어 1 3 4 2 수열의 경우 증가하는 부분 수열을 LIS 알고리즘을 사용해 구하면 다음과 같습니다.

1 -> 1 3 -> 1 3 4 -> 1 2 4

1 2 4 는 전체 수열에서 증가하는 부분 수열이 아니기 때문에 답이 될 수 없습니다.

LIS 알고리즘이 진행한 순서와 대입된 위치를 나열하면 다음과 같습니다.

1 3 4 2

0 1 2 1

이를 봤을 때, 대입된 위치를 반대로 찾아 들어간다면 증가하는 부분 수열을 파악할 수 있음을 알 수 있습니다.
