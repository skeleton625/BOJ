문제 풀이를 완료했습니다.

해당 코드는 이진 탐색(lower, upper bound) 알고리즘을 사용해 문제를 풀이했습니다.

lower_bound 함수는 해당 key 값을 찾을 경우 정렬된 배열에서 가장 첫 번째 key 값을 가진 원소를 반환하게 됩니다.

만일 key 값이 존재하지 않을 경우, key값보다 큰 가장 작은 원소를 반환합니다.

upper_bound 함수는 해당 key 값을 초과하는 첫 번째 원소를 반환합니다.

만일 찾는 key 값이 배열에 존재할 경우 upper_bound 반환 원소에서 lower_bound 반환 원소를 빼면 해당 원소들 사이의

칸 개수를 얻을 수 있어 key값의 개수를 파악할 수 있습니다.

그리고 key 값이 존재하지 않는 경우에도 upper_bound, lower_bound 반환 원소가 같은 위치에 존재하기 때문에 빼게 될

경우 0이 출력되 원하는 값을 얻을 수 있게 됩니다.
