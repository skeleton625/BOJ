백준 알고리즘 2512 문제 풀이 중입니다.

문제 풀이를 완료했습니다.

해당 코드는 이분 탐색 알고리즘을 사용해 풀이했습니다.

기본적으로 입력될 예산 중, 가장 낮은 예산은 0이므로 left를 0으로 정의하고, 입력된 최대 예산 값을 right로 정의합니다.

이 left, right 를 가지고 이분 탐색을 실햏하는데 이 때, 해당 지방에 투자할 예산으로 left, right의 중간 값인 mid를 정의

합니다.

이렇게 결정된 mid값으로 최종 소비 예산을 계산하는데 mid보다 입력된 예산이 작은 경우, 입력된 예산을 최종 소비 예산을

계산하는데 사용합니다.

이러헤 구한 최종 소비 예산이 지정된 최종 소비 예산보다 클 경우 right값을 mid-1로 정의하고 지정된 최종 소비 예산보다

작을 경우 left 값을 mid+1로 정의하니다.( left값을 증가시켜주는 이유는 공통된 소비 예산의 최대 값을 찾기 위함)

이러한 계산을 반복해 얻은 mid의 최대값이 정답이 됩니다.
